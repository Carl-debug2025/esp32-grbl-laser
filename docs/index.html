🚀 ESP32-C3 Laser Engraver

一体化微型激光雕刻机 | 开源硬件 | Web控制

✓ GPLv3 开源 ✓ ESP32-C3FH4 ✓ BLE 蓝牙 ✓ Web 控制

✨ 项目亮点

🔌

一体化设计

告别模块化组装，PCB直接可用，集成A4988驱动，接线简单

📱

Web控制端

无需安装软件，浏览器直接控制，支持Chrome Web Bluetooth

📡

BLE蓝牙

无线G代码传输，摆脱USB线缆束缚，支持实时状态监控

🎯

精确定位

106.67 steps/mm分辨率，42×42mm工作范围，适合精细雕刻

🔬

ESP32-C3FH4

RISC-V单核160MHz，内置Flash，高性能低功耗

📖

开源免费

GPLv3许可证，完整开源固件+硬件+软件，自由修改

参数 数值 说明
工作范围 42×42 mm X/Y轴最大行程
定位精度 ~0.01 mm 理论分辨率
最大速度 3000 mm/min 推荐 500-1000
步进分辨率 106.67 steps/mm 20步/圈 × 16细分 ÷ 3mm

步进驱动 A4988 × 2 16细分模式
激光功率 250-500mW PWM控制，12-bit
